Notes

In previous weeks we have been learning about arrays and functions and they are built-in objects as part of the JavaScript language.

Objects literals are created by the developers and assign values as properties of it and functions as methods. Something interesting is that a function in an object is also its property and receive the name of method like in other programming languages.

Objects are defined in the following way: const anObject = {}

Properties are key-value pairs declared in the following way: age : 36. If we have several properties, we separete them by commas like this:

const anObject = {age : 36,
name: 'Object Literal',
doSemething(){
//code for doing something when the method is called
}
}

Properties can be accessed by the dot notation or by square bracket notation like this:
anObject.age
or
anObject['age']
Square brackets notation are best when properties name are separeted by a space, something like anObject['Action performed'];

Properties and methods can be changed or removed, even when they are assigned as a const

There are many built-in functions to help us with math operations, getting dates and create dates objects.

JSON uses object literal notation to exchange information across different plaforms

It is essential to understand about DOM, because it is the organization of an HTML document and JavaScript can access the elements of HTML by using the document object and its methods to get elements by ID, Classes and Selectors and Tags name. Also we have query selectors that can helps us change style of the page. For example, I wanted to put in practice this in my notes by using querySelectorAll to get the classes that will receive an action of a JavaScript Function according to the onclick and scroll event listener that I used.

Event listeners can help us te create a more interactive webpage as we are able to combine it with DOM manipulation. Event listeners can be handled inline on a tag or in a JavaScript file or script tag. Event listener works with callbacks functions when an event happens.

Event listener also can be added or removed with addEventListener() and removeEventListener().
