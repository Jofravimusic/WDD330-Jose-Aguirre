Notes

Functions are first-class objects, this means they can be passed in the same way as every other value. They have their own properties and methods.

Functions can call itself and iterate until a condition is achieved. This is called Recursive Functions.

Callbacks are used to facilitate event-driven Asynchronous programming, because JavaScript excecute a code once at a time.

Promises are used to represent the future result of and Asynchronous operation. Promises are created with a constructor function in the following way:

const promise = new Promise( () )

Async Functions are used with the await keyword. This help to excecute when a promise or a condition is met.

Ajax is a way to make a webpage dynamic, avoiding to reload the page everytime new content need to be rendered. "Ajax" stand for: Asynchronous JavaScript and XML. Though XML was the standard used many years ago, now Ajax works is most used with json because of it native interpretation by JavaScript as an object notation of it. XML require extra work to make it render on the page, for that reason developers prefers to used json instead.

Previously, XMLHttpRequest object was widely used to work with Ajax, but now is better used the living standard of Fetch API

We can use fetch() in the following way:

fetch('https://somelink.com/here')
.then(//Something happens here)
.catch(//Here will handle the errors)
