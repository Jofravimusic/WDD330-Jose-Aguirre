Object-Oriented Programming is a style of programming. Many languages creates classes and then creates objects using those classes as instances of a Class. In JavaScript this was done exactly the same we learn last week when creating Objects Literal before ES6, because classes have properties and methods and objects in JavaScript are similiar to it. Then in ES6 was introduced the new class declaration. This is an example:

class Animal{
 constructor(name, family){
  this.name = name;
  this.family = family
 }
 run(){
  return `${this.name} is running`;
 showFamily() {
  return `${this.name}'s family is ${this.family}`;
  }
 }
}

Creating an instance of a class is like this:

const cat = new Animal('cat', 'feline');

By default, properties and methods on JavaScript are public, but we can use getter and setter like in other programming languages. We achieve this by writting properties with and underscore before the name. Example with the previous Animal Class:

class Animal{
 constructor(name){
  let _name = name;
 }
 this.setName = name => { return _name = name; }
 }
 this.getName = () => _name;
}
