Client-side and server-side validation should always be in our projects, because there would be malicious users that can affect or application even passing client-side validation.

There are two types of client-side validation

  1. Built-in form validation: This uses HTML5 form validation features like require, min, max, minlength, maxlength, pattern and type. These features are so powerfull but they are less customizable.
  2. JavaScript validation code: This option allows us to customize our validation process, but it requires to do everything
